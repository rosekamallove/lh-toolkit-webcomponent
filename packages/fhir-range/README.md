# fhir-range
The FHIR range datatype's custom element using HTML5 span tag. By default, it shows an empty  field.

### Properties of fhir-range
 * `value` - `Object`
 * `low` - `Boolean`
 * `high` - `Boolean`
 * `systemField` -`String`
 * `codeField` -`String`

  ### Typical Use:
  * With url:
  `<fhir-range url=""></fhir-range>`
  * Without url:
   `<fhir-range></fhir-range>`
